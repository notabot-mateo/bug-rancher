# Requirements

## v1 — Breeding Loop MVP

### Core Mechanics

- [ ] **CORE-01**: Player can own and view a collection of bugs
- [ ] **CORE-02**: Each bug displays its species, stats, IVs, and quality rating
- [ ] **CORE-03**: Player can select two bugs to breed
- [ ] **CORE-04**: Breeding produces an egg with incubation timer
- [ ] **CORE-05**: Hatched bug inherits genetics from parents (IV floor system)
- [ ] **CORE-06**: Player can sell bugs for currency

### Progression

- [ ] **PROG-01**: Player earns currency from selling bugs
- [ ] **PROG-02**: Player can upgrade incubation speed
- [ ] **PROG-03**: Player can unlock additional breeding slots (start with 1, max 3)
- [ ] **PROG-04**: Higher quality bugs sell for more currency

### UI/UX

- [ ] **UI-01**: Main screen shows bug collection
- [ ] **UI-02**: Bug detail view shows full stats and lineage
- [ ] **UI-03**: Breeding preview shows expected IV floors
- [ ] **UI-04**: Incubation progress visible with time remaining
- [ ] **UI-05**: Star rating (1-5) visible at a glance

### Data

- [ ] **DATA-01**: Game state persists between sessions (save/load)
- [ ] **DATA-02**: 10 bug species available from start

## v2 — Combat & Runs

- [ ] Expedition system (send bugs on timed runs)
- [ ] Combat encounters during runs
- [ ] EV training through combat
- [ ] New species discovered on runs
- [ ] Biome system (different run environments)

## Out of Scope

- Multiplayer/trading — single player focus
- Real-money transactions — not a mobile gacha
- Complex storyline — mechanics-first design
- 3D graphics — 2D pixel art only
